# Git workflow for CouponFind

- The single source of truth is the **`main`** branch. It is the default branch
  and must always contain the latest code.
- **Always commit and push directly to `main`.**
- **Do NOT create feature branches, sub-branches, or separate PR branches** for
  this repository unless the user explicitly asks for one.
- If work ever lands on another branch, merge it into `main` and keep only that
  one branch.
